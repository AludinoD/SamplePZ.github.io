Fantastic Hardcore Survival RPG Game

★★★★★

Project Zomboid is a fantastic hardcore survival RPG game for fans of Zombie games. The game may still be in early access but it is still currently be worked on. It has an incredible amount of things and interactions you can have on the world, and has some truly unforgiving survival elements. While the game does have it's bugs and issues, one of the greatest features is that the game is completely split screen couch co-op compatible. It is one of the few Steam games, ( and one of the few RPG games in general) that you can play whilst sitting next to your friend in the same room. I have fully reviewed the game on Set Ready Game, a gaming blog, but just felt the need to give the game the exposure it deserves. The Indie Stone are creating something amazing.

Great Early Access Game

★★★★☆

A great early access game with new features coming out very regularly. A very fun game to explore and, basically, survive. Do not expect to live long as the game clearly states "This is how you died". The sandbox mode also allows great tweaking for different game style for each player. Don't let yourself get fooled by "unplayed" reviews and go download the demo after you watch a couple of videos of the game.

Enough To Keep You Entertained

★★★★☆

Project Zomboid has had a checkered past at times, but it's come a long long way. It's still in Early Access after a number of years but make no mistake, it's feature full and fully playable. There's enough here to keep you entertained a long time and it's one of those atmospheric games that draws you in quite well. I would have given it a higher score only for the UI is a bit bad at times but the latest few patches have addressed it somewhat so it's going in the right direction. The devs do regular updates and are working on it still, so every month or so there's new stuff to check out, and the game is pretty configurable when it comes to starting conditions / types of zombies / loot rarity / character creation etc. I've been playing since build 27, and I think it's on build 33 now and much has been added to the game.

One Of The Best Survival Games Out There

★★★★☆

One of the best survival games out there. It's a fantastic game as-is and definitely worth the price, and yet the potential of this game is vast. Don't be deterred by the dated graphics of project zomboid. What it lacks in graphic definition it easily makes up for in gameplay and content. In fact I consider the simple graphics a boon to the game, giving ease for developers and modders to add droves of great content. The game is immersive and exciting. Playing with my wife as we explored (ran for our lives) Muldraugh, KY, I found my heart racing, and palms sweating as we were both frantically yelling across the room to eachother to coordinate our survival. If you like survival, or anything zombie-related (not mutually exclusive), BUY THIS GAME!
